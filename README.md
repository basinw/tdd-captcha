# CAPTCHA
| pattern | left operand | operator | right operand |
|---------|--------------|----------|---------------|
|    1    |       1      |    +     |      One      |
|    2    |      One     |    +     |       1       |
|         |   `1 - 9`    | `1 - 3`  |     `1-9`     |
                
Number can be `1 - 9`  
Numeric string can be `"One" - "Nine"`  
Operator can be `"+", "-", "*"`
